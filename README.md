#  수평선 기반 모션 데이터 추정

##  프로젝트 개요  
  
* 이 프로젝트는 사전 학습된 **SegFormer-b1** 모델을 활용하여 1인칭 영상 속 하늘 마스크를 생성하고, 추출된 마스크의 수평선 경계를 분석하여 영상의 모션 데이터(**Roll** 및 **Pitch**)를 추정하고 3-DoF 시뮬레이터에 연동 가능한 형식으로 저장한 후 시각화하는 프로젝트입니다.  


* 개발 인원: 3인


* 개발 기간: 2025.03 ~ 2025.06

---
  
## 주요 기능  
  
* **하늘 마스킹:** SegFormer 모델과 Optical Flow 기반 후처리를 통해 가려진 영역을 복원하며 하늘 마스크를 생성합니다.  


* **모션 데이터 계산:** 추출된 하늘 마스크의 경계선에 **3차 다항식 피팅** 및 **선형 회귀**를 적용하여 수평선을 추정하고 모션 데이터를 계산합니다.  


* **데이터 출력 및 시각화:** 처리된 모션 데이터를 시뮬레이더에 연동 가능한 형식의 엑셀(`.xlsx`) 파일로 저장하고, 그래프(`.png`)를 생성합니다.

## 문제 및 해결

### 1. 가려진 하늘 영역 복원

* **문제:**  **1인칭 영상**의 특성상 카메라 시야를 가리는 사물(Occluder) 때문에, 하늘 마스크 추정 결과가 **수평선** 분석에 필요한 수준의 정밀도**를 확보하지 못했습니다.


* **원인:** 사전 학습된 **SegFormer 모델**이 **모든 사물을 분류**하기 때문에 단순히 'sky' 클래스만 추출할 경우, 하늘을 가리는 사물(Occluder) 뒤의 영역이 하늘로 분류되지 않아 실제 하늘 영역을 추출하지 못했습니다.


* **해결:** 
    - 종류별로 하늘을 가리는 주요 사물을 **Occluder 클래스로 정의**하였습니다. 이후 기존의 하늘 마스크를 **OpenCV의 dilate로 팽창**하고 해당 영역과 **Occluder 클래스의 사물과 겹치는 부분을 'sky'로 분류**하였습니다. <br>

    - **OPEN/CLOSE 연산**을 통해 마스크 내부의 작은 구멍을 메우고 오분류 픽셀을 제거한 후 경계를 부드럽게 하여 **노이즈를 제거**합니다. <br>

    - **Optical Flow**를 계산하여 이전 프레임의 마스크와 이후 프레임 마스크를 **가중치 혼합**하여 갑작스러운 변화를 줄이고 **시간적 연속성**을 확보하였습니다.

---

### 2. 직선 피팅 안정화

* **문제:**  추정된 직선이 하늘 마스크의 곡선 전체를 반영하지 못하고 특정 부분에 지나치게 민감하게 반응하여, 최종 수평선이 **왜곡**되거나 진동**하는 현상이 발생했습니다.


* **원인:** **RANSAC** 알고리즘은 Outliers가 많은 데이터에서 강력한 성능을 발휘하도록 설계되었습니다. 이미 후처리 과정으로 **노이즈가 제거된 하늘 마스크 데이터**에 RANSAC를 적용하자 임의로 선택된 데이터에 높은 가중치를 주어 **수평선이 왜곡**되었습니다.


* **해결:** RANSAC 대신 최소 제곱법에 기반하여 데이터 **전체**의 오류를 최소화하는 직선을 찾는 **NumPy의 `polyfit` 함수**를 사용하여 **직선**을 피팅하도록 수정했습니다.

## 파일 구조  

### 1. `HorizonDetector/`

| **파일**                    | **역할**                             | **내용 상세**                                                                                                                                                                             |
| ------------------------- | ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`horizon_detector.py`** | 전체 영상 처리 흐름 제어 및 Roll/Pitch 데이터 추출 | 영상 파일을 읽고 프레임별 **하늘 마스크를 입력**받아, **3차 다항식 피팅(Curvature Correction)** 및 **선형 회귀(Roll Angle)**를 적용하여 Roll/Pitch를 계산. 계산된 데이터에 **Kalman Filter**를 적용하여 **노이즈를 제거**하며, 최종 데이터를 엑셀로 출력합니다. |
| **`sky_mask.py`**         | 딥러닝 기반 하늘 영역 추출 및 후처리              | 사전 학습된 **SegFormer-b1 모델**을 로드하여 하늘 영역의 **Semantic Segmentation** 수행. 이후 **Optical Flow** 기반의 **가려진 영역(Occlusion) 복원 후처리**를 적용하여 마스크의 정확도를 극대화합니다.                                    |
| **`draw_graph.py`**       | 분석 결과 데이터를 그래프 이미지로 변환             | 출력된 **`.xlsx` 파일**을 읽어와 **matplotlib** 라이브러리를 사용해 Roll 및 Pitch 모션 데이터의 시계열 그래프를 **자동 생성**하고 `.png` 파일로 저장합니다.                                                                         |
### 2. 입출력 및 환경 파일
| **파일/폴더**              | **역할**                            | **내용 상세**                                                         |
| ---------------------- | --------------------------------- | ----------------------------------------------------------------- |
| **`inputs/`**          | 분석 대상 원본 1인칭 영상 파일                |                                                                   |
| **`outputs/`**         | 처리된 모든 결과물(영상, 데이터, 그래프)의 Root 폴더 |                                                                   |
| **`outputs/videos/`**  | **Roll/Pitch 정보 오버레이 영상**         | 추정된 수평선과 Roll/Pitch 텍스트 정보가 실시간으로 **오버레이**된 결과 영상.                |
| **`outputs/xlsx/`**    | **모션 데이터 엑셀 파일**                  | 프레임 번호, Roll 값, Pitch 값이 기록된 **3-DoF 시뮬레이터 연동 형식**의 `.xlsx` 데이터셋. |
| **`outputs/graph/`**   | **모션 데이터 그래프 이미지**                | `draw_graph.py`를 통해 생성된 Roll/Pitch 데이터 시각화 `.png` 파일.             |
| **`requirements.txt`** | 프로젝트 실행에 필요한 Python 라이브러리 목록      |                                                                   |

## 설치 및 실행 방법  
### 1.  **프로젝트 클론:**  
터미널을 열고 원하는 위치에서 다음 명령어를 실행하여 프로젝트를 클론합니다.  
```bash  
  git clone https://github.com/Cha0910/Motion-Data-Horizon.git  
  ```  
### 2. 환경 설정  
  
`requirements.txt` 파일을 사용하여 필요한 모든 Python 라이브러리를 설치합니다.  
  
```bash  
asdpip install -r requirements.txt
````  
  
### 3. 실행 방법  
  
1.  **원본 영상 준비:** 처리할 영상 파일들을 **`../inputs/`** 폴더에 복사합니다.  
  
2. **스크립트 실행:** `horizon_detector.py`를 실행합니다.  
  
  
```bash  
python HorizonDetector/horizon_detector.py
```  
  
### 4. 사용자 입력  
  
실행 중 다음과 같은 입력이 필요합니다.  
  
1.  **영상 환경 선택:** 영상의 환경에 맞춰 하늘을 가리는 객체 복원 유형을 선택합니다.  
      * `1`: 드론/비행기  
      * `2`: 레이싱  
      * `3`: 보트  
  

2.  **Pitch Scale 입력:** Pitch 데이터의 민감도를 조정할 배율을 입력합니다. (기본값: 200)  
  
### 5. 결과 확인  
  
스크립트 실행이 완료되면, 모든 영상 처리와 엑셀 저장이 완료됩니다.  
  
그래프 이미지를 생성하려면, 다음 명령어를 실행합니다.  
```bash  
python HorizonDetector/draw_graph.py
```  
결과물은 **`../outputs/`** 폴더 내 각 하위 폴더에 저장됩니다.
